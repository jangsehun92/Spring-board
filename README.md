# Spring-board
Spring Framework를 사용하여 게시판을 구현한다.

## Front-end
* Bootstrop3
* jQuery3

## Back-end
* Language : JAVA 1.8
* WEB Server : Apache
* WAS Server : Tomcat 8
* Framework : Spring 5.1.3
* build tool : Maven 3.5.1
* ORM : mybatis 3.4.6
* DB : Oracle 11g

## 기능 목록
* [spring security5] 회원가입
* [spring security5] 로그인/로그아웃
* 이메일 인증 
* Ajax를 이용한 게시글 CRUD 
* 페이징 처리 
* 게시글 관련 REST API  
* 댓글 CRUD
* 추천 기능
* 검색 기능
